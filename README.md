# MNIST from scratch

A hands-on route from inspecting MNIST data and manually implementing a neural
network to training, debugging, and eventually extending models with PyTorch.

## Learning artifacts

- `mnist-from-scratch.ipynb` — inspect MNIST and build the NumPy foundations.
- `main.py` — one complete manual forward/backpropagation/update pass.
- `mnist-using-pytorch.ipynb` — train an MNIST MLP with PyTorch.
- `pytorch_hands_on_playground.ipynb` — small experiments spanning tensors,
  autograd, modules, training, embeddings, and attention.
